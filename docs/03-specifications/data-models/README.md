# Data Models

This directory contains data models and schemas for the MedMonitor system.

## Models
1. **User** - System users (patients, doctors, admins)
2. **Patient Profile** - Patient medical information
3. **Health Metric** - Health measurements (heart rate, blood pressure, etc.)
4. **Alert** - System-generated alerts
5. **Device** - Medical devices

## Data Storage
- **PostgreSQL** - Relational data (users, profiles)
- **MongoDB** - Time-series health data
- **Redis** - Caching and session storage

## Schemas
- JSON Schema definitions
- Database migration scripts
- Data validation rules
