# Quality Attributes

This directory documents the quality attributes and non-functional requirements for MedMonitor.

## Key Attributes
1. **Security** - Data protection and access control
2. **Performance** - Response times and throughput
3. **Reliability** - System availability and fault tolerance
4. **Scalability** - Handling growth in users and data
5. **Usability** - User experience and accessibility
6. **Maintainability** - Code quality and ease of changes

## Metrics
- Response time: < 200ms for 95% of requests
- Availability: 99.9% uptime
- Data retention: 7 years for medical data
- Backup frequency: Daily full, hourly incremental
